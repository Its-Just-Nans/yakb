## Sources

- [PostGIS](https://postgis.net/) - a spatial database extender for PostgreSQL object-relational database
- [MBTiles](https://github.com/mapbox/mbtiles-spec) a [Mapbox](https://www.mapbox.com/) specification for storing tiled map data in SQLite databases
- [PMTiles](https://github.com/protomaps/PMTiles) is an open specification for single-file tile pyramids built on compressed Hilbert ordering and queryable via HTTP Range Requests

## Servers

Servers produces map from sources and serve them to clients.

- https://wiki.openstreetmap.org/wiki/Tile_servers
- https://github.com/maplibre/martin

## Client

- https://leafletjs.com/ - an open-source JavaScript library for mobile-friendly interactive maps. Renders tiles as images.
- https://maplibre.org/maplibre-gl-js - a JavaScript library that uses WebGL to render interactive maps from vector tiles and Mapbox styles
