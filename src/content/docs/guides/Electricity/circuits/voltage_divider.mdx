A voltage divider is a circuit that divides the input voltage into two or more parts.

**Formulas for voltage division:**

<Flexed>

```math
U_{\textcolor{blue}{2}} = E * \frac{R_{\textcolor{blue}{2}}}{R_1 + R_2} = \frac{R_{\textcolor{blue}{2}} * E}{R_1 + R_2}
```

```math
U_{\textcolor{blue}{1}} = E * \frac{R_{\textcolor{blue}{1}}}{R_1 + R_2} = \frac{R_{\textcolor{blue}{1}} * E}{R_1 + R_2}
```

</Flexed>
