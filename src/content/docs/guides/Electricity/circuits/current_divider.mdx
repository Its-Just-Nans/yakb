A current divider is a circuit that divides the input current into two or more parts.

**Formulas for current division:**

<Flexed>

```math
I_{\textcolor{red}{2}} = \frac{R_{\textcolor{red}{1}} * I}{R_1 + R_2}
```

```math
I_{\textcolor{red}{1}} = \frac{R_{\textcolor{red}{2}} * I}{R_1 + R_2}
```
